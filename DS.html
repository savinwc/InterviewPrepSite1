DataStructure :

Data structures are ways to organize and store data in a computer so that it can be accessed and modified efficiently. They are fundamental to computer science and programming, as they help manage large amounts of data in a way that allows for efficient processing and retrieval. Different data structures are suited for different types of tasks, depending on how you need to access or manipulate the data.
