List in Java

The List interface is found in java.util.package and inherits the Collection interface. It is a factory of the List Iterator interface. Through the List Iterator, we can iterate the list in forward and backward directions. The implementation classes of the List interface are ArrayList, LinkedList, Stack, and Vector.

ArrayList Overview

  • ArrayList is a Java class implemented using the List interface.
  • It is a dynamic array and its size automatically increases when we add or remove elements.
  • Basic functionality when the array becomes full:
    • Creates larger memory on the heap (e.g., double size).
    • Copies current elements to the new memory space.
    • Adds the new element into the newly allocated space.
    • Deletes old memory space.
  • ArrayList stores integer-type objects.

ArrayList Constructors

Constructor Description Example
ArrayList() Builds an empty array list. ArrayList arr = new ArrayList();
ArrayList(Collection c) Builds an array list initialized with elements from collection c. ArrayList arr = new ArrayList(c);
ArrayList(int capacity) Builds an array list with the specified initial capacity. ArrayList arr = new ArrayList(N);

ArrayList Methods

Method Description
add(int index, Object element) Inserts a specific element at the specified index in the list.
add(Object o) Appends a specific element to the end of the list.
addAll(Collection C) Appends all elements from the specified collection to the end of the list.
addAll(int index, Collection C) Inserts all elements starting at the specified index from a collection into the list.
clear() Removes all elements from the list.
clone() Returns a shallow copy of the ArrayList.
contains(Object o) Returns true if the list contains the specified element.
ensureCapacity(int minCapacity) Increases capacity to ensure the list can hold at least the specified number of elements.
forEach(Consumer<? super E> action) Performs the given action for each element until all elements are processed or an exception is thrown.
get(int index) Returns the element at the specified index in the list.
indexOf(Object o) Returns the index of the first occurrence of the element or -1 if it's not found.
isEmpty() Returns true if the list is empty.
lastIndexOf(Object o) Returns the index of the last occurrence of the element or -1 if not found.
listIterator() Returns a list iterator over the elements in this list in proper sequence.
remove(int index) Removes the element at the specified position in the list.
remove(Object o) Removes the first occurrence of the specified element from the list, if present.
size() Returns the number of elements in the list.
